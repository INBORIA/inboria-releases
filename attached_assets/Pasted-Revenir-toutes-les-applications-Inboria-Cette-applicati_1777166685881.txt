Domaines
Les domaines déterminent les autorisations dont dispose votre application pour accéder à des données ou réaliser des actions dans HubSpot. Les personnes qui installent votre application doivent approuver les domaines qu'elle demande. Il est vivement recommandé d'exiger le moins de domaines possible pour le fonctionnement de votre application. En savoir plus
